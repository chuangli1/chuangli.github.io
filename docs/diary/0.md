# 回家的一些感想
## 阔别两年的家乡
转眼间，已经将近两年没回家了，今年过年经过深思熟虑还是决定回家，回家对于我来说是一种选择，在这世间转眼已经有了24年了，可是问我就是没有所谓的思乡之情，对于老家的亲人没有什么思念<br/>
今年是牛年，我的本命年。本打算是2月9号回的，后来上海疫情严重，又传来了国务院说返乡要核酸监测和隔离的消息，身边的同学都争先恐后的提前回去了，所以我把票改签到了4号。在所有同学都回去之后的一个星期，突然觉得好无聊，2月2号预约了学校的集体核酸检测，检测完本想在实验室摸鱼，然而发现不太想学习了，遂决定立即返家。。。
## 参加婚礼 
邻居小孩结婚，新郎是和我一块长大的，不过已经多年未见，我时隔多年再次在农村参加婚礼，实在是有点不适应，也无法和这边的人共情，新郎当兵两年后结婚，后面和河南一所二本院校读农业专业，但是好像在一系列复杂的情境中决定结婚，本来说要和他玩玩谈谈心之类的，但是新郎比较忙，自己也觉得不知道聊什么，遂作罢。。。隔日晚上新郎来我家玩，和我聊起了为什么结婚，农村的魔幻现实主义啊。。。
## 看望老人
回家第一天就得知我姨姥住院了，在县里面的人民医院，不久又打电话说回来了，和妈妈一起去探望，是劳累导致的骨折，需要卧床一个月；之后的几天，有去看望了大腿骨折卧床近一年的姥姥，在养老院的精神分裂的奶奶；家里的老人都老了，自己也长大了，百感交集。
## 渐渐醒悟
家乡对于一个人是什么意义，我一直不太明白，可能与从小对家乡的排斥有关。小时候家庭贫困，农村人的小农思想注定了贫困家庭的小孩在村里受人冷眼，好在脑子比较好使，学习成绩很好，在老家也算闻名，通过高考也算是改变了命运。对家乡人和父母的小农思想，什么官本位之类的思想一直嗤之以鼻，和父母也有很多矛盾；可能时光真的可以改变一个人吧，这次回家渐渐理解的父母，渐渐理解了一个人的不容易，父母年纪也大了，还在外拼命挣钱，希望他们可以一直身体健康，自己可以早日挣很多很多的钱，改变他们的生活。。
