# 回家的一些感想
## 阔别两年的家乡
转眼间，已经将近两年没回家了，今年过年经过深思熟虑还是决定回家，回家对于我来说是一种选择，在这世间转眼已经有了24年了，可是问我就是没有所谓的思乡之情，对于老家的亲人没有什么思念，父母将这一点归我独立的性格，我认为和我的成长经历有关。<br/>
今年是牛年，我的本命年。本打算是2月9号回的，后来上海疫情严重，又传来了国务院说返乡要核酸监测和隔离的消息，身边的同学都争先恐后的提前回去了，所以我把票改签到了4号。在所有同学都回去之后的一个星期，突然觉得好无聊，2月2号预约了学校的集体核酸检测，检测完本想在实验室摸鱼，然而发现不太想学习了，遂决定立即返家。。。
